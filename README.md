# Passion-culminating - (name)

## General info

A two player game where players have to survive as long as possible from projectiles coming from 4 directions.
Players share a barrier to defend themselves with, but since they share a block the oldest block will disapear
when a new block is placed. after some times passes extra blocks will be added in up to 5 blocks.

players will also be able to move around the grid.
